我用树莓派搭建的网站

空间小不稳定

如果哪天树莓派坏掉那不是数据就没有了

怎么办

在网上找的办法，可以使用python的bypy库实现百度云的上传功能

首先为python安装requests、bypy库 


pip install requests
pip install bypy
之后使用bypy命令 
bypy info

然后会得到一个网址，通过这个网址使用账号授权，得到一个授权码


将这个码输入然后继续回车，就算是授权成功

进入需要上传的目录

就可以使用命令上传同步了 

bypy upload   //同步到百度云

这样就将文件夹数据全部同步到百度云的/我的应用数据(apps)/bypy中


之后我使用宝塔面板的计划任务进行备份，并且同步到百度云

因为宝塔面板的用户不知道，之前的授权也没有作用到宝塔面板

怎么办-_-&

使用宝塔面板计划任务，运行shell，输入授权码 


echo "授权码" | bypy info
然后授权成功
